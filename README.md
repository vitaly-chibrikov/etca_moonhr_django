# ETCA Moon HR

Welcome to the repository for "ETCA Moon HR", a science fiction video game that explores life in a thriving Moon colony. 
This project is created using Python Django and powered by AI to deliver an immersive lunar experience.

## About the Project

"ETCA Moon HR" is an interractive novel game set in a fictional lunar colony of the ExtraTerrestrial Colonies Agency (ETCA). 
In this futuristic world, you'll navigate life on the Moon, make critical decisions, and explore the challenges and adventures of a thriving lunar society.
